--- layout: post date: 2017-07-07 title: 在ubuntu16 安装 phpamyadmin ---

第一步:从ubuntu仓库中下载我们的phpmyadmin

 $ sudo apt-get install phpmyadmin php-mstring php-gettext

安装的过程中会询问你几个问题，对于弹出一个server选择框，你可以看到apache2是高亮选中的，但是你必须按一下空格，它才真的选中 这一点要注意。之前可能会询问是否使用dbconfig-common来启动数据库，选择yes，然后确认密码。

整个的安装过程实际就是把phpMyAdmin Apache的配置文件放到了/etc/apache2/conf-enabled目录中，在这自动加载读取，我们唯一要做的事情 就是启动php的mcrypt和mbstring的扩展。

 $ sudo phpenmod mcrypt 
 $ sudo phpenmod mbstring

重启我们的额apache2的服务:

 $ sudo systemctl restart apache2
