检查如下

我的原来是没有fcitx的，是后来安装上的。上面部分是语言选择，这个其实和输入法没有什么关系。

第二步  安装fcitx

$ sudo add-apt-repository ppa:fcitx-team/nightly
$ sudo apt-get update
$ sudo apt-get install
$ sudo apt-get install fcitx-config-gtk
$ sudo apt-get install fcitx-table-all
$ sudo apt-get install im-switch

检查一下是否安装成功

出现这个fcitx-configuration这几个就行了，等会这个fcitx-configuration才要真要配置搜狗等输入法。

第三步  下载搜狗输入法

搜狗输入法 for linux

安装过程中出现下面的错误，不用管

到 系统设置->语言支持，将键盘输入法系统设置为fcitx,其他的像上面的language就不用管了，其实就是系统语言设置

最重要的来了，设置fcitx输入系统后，最重要的步骤来了，重启系统或者称为注销
搜索fcitx然后点击fcitx-configuration 进入下面的画面

你第一次打开的时候会没有sougou输入只有默认的英文输入，也可能什么都没有。所以要添加，如下图，点击+号弹出

搜索到sougou 输入法，然后点击add，添加进去。如果你没有找到搜狗输入法。那么久把红色勾选去掉。然后再找。重启电脑之后你就可以ctrl+space自由切换了。

大功告成!
