问了Sql语句查询问题
有一个表 t

|MONTH| SALARY  |
|--|--|
|  1|  1000|
|2|1200|
|3|1300|
然后通过sql查询语句生成下面的的表
|MONTH| SALARY  |total
|--|--|--|
|  1|  1000| 1000|
|2|1200| 2200|
|3|1300| 3500|

可以通过嵌套语句实现
```
SELECT month, salary
	(SELECT sum(salary) 
	FROM t as t1
	WHERE t1.month <= t.month
	) as total
FROM t;
```
